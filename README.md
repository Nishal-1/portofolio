## 🚀 Deployment

### Option 1: GitHub Pages (Recommended)

1. Create a new repository named `portfolio` or `username.github.io`
2. Push the portfolio files to the repository
3. Enable GitHub Pages in repository settings (Settings > Pages > Deploy from main branch)
4. Your portfolio will be live at `https://username.github.io/portfolio` or `https://username.github.io`

### Option 2: Netlify

1. Connect your GitHub repository to Netlify
2. Set build command: (leave empty for static files)
3. Set publish directory: `./` (root directory)
4. Deploy button will automatically trigger on pushes

### Option 3: Vercel

1. Import your GitHub repository to Vercel
2. Vercel auto-detects the static site
3. Deploy with zero configuration

### Option 4: Self-Hosted

1. Upload `index.html` and any assets to your web server
2. Ensure your domain is configured to point to the server
3. Access via your custom domain

## 🎨 Customization

### Change Colors

Edit the CSS gradient in the `<header>` section:

```css
background: linear-gradient(135deg, #1e3c72 0%, #2a5298 100%);
border-bottom: 4px solid #00d4ff;
```

Replace `#1e3c72`, `#2a5298`, and `#00d4ff` with your preferred colors.

### Update Profile Photo

Replace the base64 image data in the `<img>` tag with your photo:

```html
<img src="data:image/jpeg;base64,YOUR_BASE64_HERE" alt="Nishal P" class="profile-img">
```

Or use a direct URL:

```html
<img src="path/to/your/photo.jpg" alt="Nishal P" class="profile-img">
```

### Modify Contact Information

Update the contact section in the header:

```html
<div class="contact-info">
    <span>📱 +91 XXXXXXXXXX</span>
    <a href="mailto:your-email@example.com">📧 Email</a>
    <a href="https://github.com/your-username" target="_blank">💻 GitHub</a>
</div>
```

### Add More Projects

Duplicate a project card and update the content:

```html
<div class="project-card">
    <h3>Project Title</h3>
    <div class="project-tech">
        <span class="tech-tag">Technology</span>
        <span class="tech-tag">Stack</span>
    </div>
    <p class="project-desc">Project description...</p>
    <a href="https://github.com/..." target="_blank" class="project-link">View on GitHub →</a>
</div>
```
